# AI Coding Plan 对比工具

> 国内主流 AI 厂商 Coding Plan 套餐全面对比，助你选择最适合的方案

## 📖 简介

本项目提供国内主流 AI 平台的 Coding Plan（代码助手订阅）套餐对比，包括火山引擎方舟、阿里云百炼、Kimi、智谱 AI、MiniMax、百度千帆等厂商的多档套餐信息。

通过直观的表格展示，帮助开发者快速了解不同套餐的价格、支持的模型、请求限额、权益等内容，做出最优选择。

## ✨ 功能特点

- 📊 **全面对比** - 覆盖 6 大主流 AI 厂商的 19 个套餐
- 💰 **价格透明** - 清晰展示首月、包月、包季、包年价格及折扣
- 🔍 **智能筛选** - 支持按平台、价格区间、模型进行筛选
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 🎨 **现代化 UI** - 简洁美观的界面设计，提升浏览体验
- ⚡ **实时更新** - 数据通过 CSV 文件管理，易于维护和更新

## 📌 数据说明

### 价格计算规则

- **首月价格**：各平台新用户首月优惠价格
- **连续包月**：每月自动续费价格
- **连续包季**：每 3 个月自动续费价格，通常有折扣（显示原始价格对比）
- **连续包年**：每年自动续费价格，折扣力度最大（显示原始价格对比）

### 模型说明

各平台支持的主要模型：
- **智谱 AI**：GLM-4.7, GLM-5
- **Kimi**：Kimi K2.5, Kimi K2, Kimi K2 Thinking
- **MiniMax**：MiniMax M2.5
- **阿里云百炼**：Qwen 3.5, MiniMax M2.5, GLM-5, Kimi-k2.5
- **字节火山方舟**：Doubao-Seed-2.0-Code, Kimi-K2.5, GLM-4.7, Deepseek-V3.2
- **百度千帆**：GLM-5, Kimi-K2.5, MiniMax-M2.1, DeepSeek-V3.2

### 请求限额说明

- **5小时请求数**：单个 5 小时时段内的请求次数限制
- **每月总请求数**：每月累计可使用的请求次数
- **未公开**：部分平台未公开具体限额
