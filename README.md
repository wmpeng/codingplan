# AI Coding Plan 对比工具

> 国内主流 AI 厂商 Coding Plan 套餐全面对比，助你选择最适合的方案

## 📖 简介

本项目提供国内主流 AI 平台的 Coding Plan（代码助手订阅）套餐对比，包括字节·方舟、阿里百炼、Kimi、智谱 AI、MiniMax、百度千帆等厂商的多档套餐信息。

通过直观的表格展示，帮助开发者快速了解不同套餐的价格、支持的模型、请求限额、权益等内容，做出最优选择。
